# LXD Portfolio

A clean, single-page portfolio site for Learning Experience Design roles.

## Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `lxd-portfolio`)
2. Push this code:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/lxd-portfolio.git
   git push -u origin main
   ```
3. Go to **Settings → Pages** in your repo
4. Under **Source**, select `main` branch and `/ (root)`
5. Your site will be live at `https://YOUR-USERNAME.github.io/lxd-portfolio/`

## Customize

Replace all `[bracketed placeholders]` in `index.html` with your info.
